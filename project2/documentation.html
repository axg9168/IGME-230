Project Requirements

Main Requirements

  • Application will utilize a web service.
  • Application will save the last term searched by the user using local storage.
  • Require a minimum of three controls the user can use to filter and display results.
  • No JavaScript errors or exceptions thrown by the application.

Design & Interaction

  • Pleasing graphic design.
  • Widgets are well labeled and follow interface conventions.
  • Users should be able to figure out how to use the app with minimal instruction.
  • User errors must be handled gracefully.
  • Users must know what state the app is in at all times.

HTML/CSS & Media

  • Valid HTML5
  • Valid CSS
  • Most CSS is in an external style sheet.
  • Use HTML5 semantic and structural elements where practical.
  • Images are properly optimized for Web delivery.

Code Conventions

  • All code is an external JavaScript file - inline event handlers are not allowed
  • let and const must be used to declare variables (no var!)
  • querySelector() and querySelectorAll() must be used for DOM traversal
  • Don't Repeat Yourself. Repeated blocks of nearly identical code must be factored out and placed in a separate function.
  • Variable and function names must begin with a lowercase letter.
  • Well-commented code. Each and every function gets a comment indicating what it does.
  • Delete or comment out any console.log() calls.

Resources Used

  • API Used: PokeAPI
  • Fonts Used: Patrick Hand SC and Josefin Sans: Google Fonts
  • Pokemon and Type images: Serebii
  • Simulating Button Click in JavaScript: Stack Overflow
